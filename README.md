Le travail demandé consiste à simuler la convergence du modèle BOPM vers le modèle BSM en
se basant sur les éléments du cours, du logiciel MATLAB et en se servant des données suivantes:
Stock Price 50

r 5.00%
u à calculer
d à calculer
